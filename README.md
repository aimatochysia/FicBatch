<h1>FicBatch</h1>
<h2>A cross-platform AO3 browser & reading experience ‚Äî rebuilt for power users.</h2>
<p>FicBatch is a desktop and mobile app designed to make reading Archive of Our Own (AO3) fiction faster, smoother, and far more customizable.
It acts as a specialized AO3 browser with an integrated reader, advanced library tools, and optional background sync so you never miss an update.</p>

<h2>Features:</h2>

<h3>üìñ Reading Experience</h3>
<ul>
  <li>Embedded AO3 browsing with custom enhancements</li>
  <li>Dedicated reader with autosave & chapter navigation</li>
  <li>Configurable reader mode: Prefer Online, Prefer Downloaded, Always Online, or Always Downloaded</li>
  <li>Offline reading support with downloaded works</li>
  <li>Cross-format chapter detection (works with both online and downloaded HTML)</li>
  <li>Font size, scroll speed, and theme customization</li>
  <li>Reading progress synced between online and offline modes</li>
</ul>

<h3>üìö Library Management</h3>
<ul>
  <li>Local library system for saving and organizing works</li>
  <li>Category-based organization with custom categories</li>
  <li>Batch import: paste multiple AO3 URLs or work IDs to import at once</li>
  <li>Download works for offline reading</li>
  <li>Batch download by category with auto-download option</li>
  <li>Export/Import library as JSON for backup and cross-device sync</li>
</ul>

<h3>üîÑ Sync & Updates</h3>
<ul>
  <li>Background sync on mobile to detect new chapters</li>
  <li>In-app sync on Windows (requires app to be running)</li>
  <li>Manual sync available on all platforms</li>
  <li>Update notifications for tracked works</li>
</ul>

<h3>üìä History & Tracking</h3>
<ul>
  <li>Reading history with timestamped access logs</li>
  <li>Chapter-based reading progress tracking</li>
  <li>Percentage-based progress for font-size independence</li>
</ul>

<h3>‚öôÔ∏è Settings & Customization</h3>
<ul>
  <li>Light/Dark theme support</li>
  <li>Configurable library grid columns</li>
  <li>Auto-sync interval and network preferences</li>
  <li>Storage info and management</li>
</ul>
